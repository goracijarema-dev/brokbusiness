# Брок Бізнес

Готовий статичний лендинг для GitHub Pages. Нічого збирати не потрібно: файли можна одразу завантажувати в репозиторій.

## Файли

- `index.html` - структура сайту та тексти.
- `styles.css` - адаптивний преміальний дизайн, snap-scroll, анімації.
- `script.js` - модалка контактів, лічильники, 3D-tilt, живий графік.
- `.nojekyll` - службовий файл для GitHub Pages.

## Публікація на GitHub Pages

1. Створи репозиторій на GitHub.
2. Завантаж файли з цієї папки в корінь репозиторію: `index.html`, `styles.css`, `script.js`, `.nojekyll`.
3. Відкрий `Settings` -> `Pages`.
4. Обери `Deploy from a branch`, гілку `main`, папку `/root`.
5. Збережи - через кілька хвилин сайт буде доступний за посиланням GitHub Pages.
